## ADHD Optimization Patterns

### Executive Function Support

**Task Initiation:**
- Break large tasks into 15-minute chunks
- Provide "first step" explicitly
- Use urgency appropriately (not everything is urgent)
- Clear action verbs (Call, Email, Buy, Schedule)

**Working Memory:**
- Capture everything immediately
- Context included in every task
- Visual indicators for priority
- Recurring task templates

**Time Blindness:**
- Automatic time estimates on every task
- Multiple reminder stages (15min, 5min, 1min)
- Travel time included in appointments
- Buffer time between tasks

**Attention Management:**
- Limit choices (3 options maximum)
- Clear visual hierarchy
- Consistent categorization
- Hyperfocus protection (break reminders)

### Energy Management

**Daily Patterns:**
- Morning (7-9am): High energy, complex tasks
