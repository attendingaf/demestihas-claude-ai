# EA-AI Response Templates

## Email Templates

### Professional Reply
```
Why This Works: [Reasoning]

[Greeting],

[Direct point - no fluff]

[Action items if needed]

[Clear next step]

Best,
[Name]
```

### Quick Decline
```
Thanks for thinking of me. This doesn't align with current priorities.

Best,
[Name]
```

## Task Breakdown Template

### ADHD-Optimized Format
```
Task: [Main Task]
Total Time: [Duration]
Energy Required: [High/Medium/Low]

15-Minute Chunks:
– Chunk 1 (0-15 min): [Specific action]
– Chunk 2 (15-30 min): [Specific action]
– Chunk 3 (30-45 min): [Specific action]

Best Time: [Morning/Afternoon/Evening]
Prerequisites: [What's needed before starting]
Success Metric: [How to know it's done]
```

## Calendar Conflict Template

### Multi-Calendar Check
```
Conflict Detected:
– Calendar 1: [Event] at [Time]
– Calendar 2: [Event] at [Time]

Travel Time Needed: [X minutes]
Resolution Options:
1. Reschedule [Event A]
2. Delegate [Event B]
3. Virtual attendance for [Event]

Recommended: [Option with reasoning]
```

## Decision Framework Template

### Options Analysis

```
Decision: [What needs deciding]

– Pros: [Benefits]
– Cons: [Drawbacks]
– Risk: [Low/Medium/High]
– Timeline: [How long]

Option B: [Description]
– Pros: [Benefits]
– Cons: [Drawbacks]
– Risk: [Low/Medium/High]
– Timeline: [How long]

Recommendation: [Choice] because [reasoning]
Next Step: [Immediate action]
```
